React Native UI学习

StyleSheet

StyleSheet类似于CSS样式表的抽象。内部定义多个样式对象，样式对象的属性使用逗号分隔，像素或者是字体大小不需要带符号(对于Android会自动转换为dp，对于ios会自动转换为pt)

使用

注意点：

  • stylesheet不要定义在render中，独立出来，使得代码清晰。
  • 需要对样式做到见名知意，style尽量与他所修饰的component有关
  • 创建的样式表通过id引用他，而不是每次都创建，对于样式表而言，他只会创建一次。
    方法(都是static的)：
  • create(obj): 创建一个样式，例如就像上面的
  • flatten:他的作用是把一个数组的样式变为一个独立的合并的样式，他可以用来查询id，该id是通过StyleSheet.register返回的。比如

属性：

  • hairlineWidth: 定义平台上的细线的宽度，可以作为两个组件之间的边界划分。他始终是一个整数屏幕像素密度，并且符合当前平台最细的线的标准。但是我们不能把它视为常量，因为不同平台他的值是不一样的。

  • absoluteFil:用于创建绝对定位和0定位的覆盖，可以用于方便和减少重复样式。

  • absoluteFillObject:有时候我们可能使用absoluteFIll，但是absoluteFillObject可以在StyleSheet中用来创建一个定制入口

flexbox

可以看这里：布局样式属性
Flexbox 是一种布局模式，可以使用flexbox规则约束子组件布局。他可以达到在不同屏幕尺寸提供一致的布局结构。(跟Android LinearLayout有点类)。他的工作原理和CSS3的基本一样，只是少了一些属性。
属性：

  • flexDirection：决定布局的主轴，从而决定子元素的方向排列方向。

结果和图：

  • JustifyContent：设置沿着主轴的排列方式。他不可以更改元素的排列顺序，只是在主轴上的排列坐标变化。

实例和解析和结果图

  • flexWrap
    他定义父组件是否允许子组件多行排列，默认是不行的

例子和图

  • alignItems
    他决定子元素在次轴的排列位置。

例子和图：

  • alignSelf
    他决定了子元素在父元素次轴上的排列方式，他会覆盖父元素的alignItems的值，他的默认值是auto

例子和图

  • flex
    他类型与Android的weight，权重。他是一个整数，而且是一个正数。主要作用是均分主轴的长度。比如我们不定义主轴各个组件的宽度，而是直接标明每个组件的flex是多少,比如下面例子，上面一个view跟下面一个View占的比例是：2:1，上面的占据了2/3，下面1/3

普通属性

  • width:宽度
  • height:高度

  • magin:外边距，包括的属性有

  • maxWidth:最大宽度

  • minWidth:最小宽度
  • maxHeight:最大高度
  • minHeight:最小高度
  • padding:内边距
    他跟margin属性一样，把margin换成padding既可。
  • position:他跟Css的position类型
    值有：relative(默认，相对定位)和absolute(相对于父容器的绝对定位)。
    absolute可以使得子元素相对于父容器一定数量的像素位置。
    reletive是相对
  • left,right,top,bottom，这四个是设置子元素相对于父元素的距离，他跟position的值相关。
