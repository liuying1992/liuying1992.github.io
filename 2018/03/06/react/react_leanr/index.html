React基础

他的特点是：

  1. 作为UI，在MVC中作为View层的存在。
  2. 虚拟DOM，实现了优化视图的渲染和刷新。react不直接操作DOM，而是通过构造一个虚拟的DOM，跟上一次的DOM树进行对比，得到差异将需要变化的部分进行刷新。
  3. 组件化，可以封装起来作为有独立功能的UI部件存在，实现复用高效率开发。
  4. 他实现了单向的数据流，相对于传统的数据绑定，React更加灵活，便捷。他是单向绑定的。

常见有三种：

  1. 内联样式
  2. 对象样式
  3. 选择器样式
    React与Html中设置样式的书写有格式区别：
  4. Html以分号结尾;，React以逗号结尾,
  5. Html中key-value都不加引号，React中，输入JS对象，key的名字不能出现-，需要 以驼峰命名法，如果value为字符串，需要加引号。
  6. Html中，如果value是数字，需要带单位，React不需要带单位。
  7. html中的class为指定选择器，在React中使用className替换。同样，html样式的for在React中使用htmlFor表示。

Props

对于Props，他是只读的，纯净的，对于所有的React Component来说，他们必须是纯净的function，不可以对Props做出修改。
props是组件本身的属性，一般用于嵌套内外层的组件中，负责传递信息(通常是由父组件向子组件传递)在最新版本中，是通过参数添加的形式写入，props对象中的属性和组件的属性是一一对应的。
