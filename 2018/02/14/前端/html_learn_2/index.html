表单

表单可以把浏览器用户输入的数据传递到服务器端，相当于Android的一些输入。他的语法是

  • from标签是成对出现的
  • action是指请求地址，对应服务器的一个接口
  • method也请求的方式，比如post/get/put等

input

  • 对于input标签，他是与用户进行交互的，给用户输入信息的一个控件。type属性是表明可以输入的类型，text为文本类型，passwor为密码，输入的时候无法看见的，sumbit为提交，相当于提交该表单，reset为重置该表单的input。name是标识改input的，一般跟后台对应需要的字段名称一样，value是默认的输入

textarea

  • input是单行输入的，需要使用多行输入，则需要使用textarea，在textarea的标签之间可以输入多行。比如

checkbox

他们也是使用input来表示，只是通过type区分

  • radio是单选

  • checkbox是多选

  • value是该input所代表的值

  • name是表示，跟后台接收该字段的名称对应
  • checked表示默认是否选中，对于radio，假如多个，每一个都是true，则只有最后一个生效。
  • 对于同一组的单选box，name需要一致。
