前端基础标签

  • <h1>是标题
    他还有多级标签，一共有h1,2,3,4,5,6中，重要性依次递减。
  • <p>是段落，他们之间是换行的

  • <image>是图片

  • <head>是头部
    head可以有的元素有

  • <body>是内容
    body的内容有

  • <html>是用来囊括整个文件的

  • HTML 标签不区分大小写，但是建议都小写

  • HTML的注释是

  • 强调语气<strong>和<em>标签，可以混合使用
    是斜体，是粗体

  • 为文字设置单独样式，使用<span>标签，他需要先进行声明，在head之前假如style标签，相等于一些全局的样式。例如

  • 引用使用<q>，他是短文本引用，长文本引用，比如一个段落，则是使用<blockquote>

  • 换行 <br/>直接闭合标签就行<br />则是html1.0写法，<br>是html4.0的写法
  • 空格  
  • 水平横线 <hr/> 他也是有html1.0和4.0，目前还是会使用html1.0的居多。
  • 地址标签<address>，默认是斜体的
  • 一行代码使用<code>标签，多行代码使用<pre>标签，在pre标签中，空格和换行不用使用特定的那些html标签表明，他会自动保留。
  • 列表，没有顺序的列表使用<ul>-<li> <ul>是组，<li>是item
  • 有序列表，他是使用<ol>-<li> <ol>是组，<li>是item

  • 容器，可以把独立的部分逻辑划分出来，放在一个div标签中。div可以使用命名来区分。使用id标识命名，例如

  • 表格，使用table属性，包括table-tr-th-td table闭合标识一个完整的表格，tr闭合标识行，th闭合标识一个独立item,用在第一行，td用在一般行例如

对于表格，还可以使用<tbody>如果不加入<thead><tbody><tfooter>，table表格会加载完成之后才显示，加上上面的表格结构，tbody包含行的内容下载完成优先显示，不必等待表格加载结束，如果表格很长，可以使用tbody分段，一部分一部分显示。（还不理解tbody和tfooter，thead的作用）
使用css为表格添加样式，比如边框。

还可以给表格定义一个摘要标题，使用summary，属于table的修饰元素，它主要是给搜索引擎读的，不会显示在浏览器中。caption会显示在table的上方。

  • 链接标签，使用 <a />比如

当我们需要在新页面打开该网址的时候，需要添加一个target属性，它默认是在当前窗口打开的。

  • 网页链接到email。在<a>标签中，使用mailto。可以通过类似URL的形式，拼接subject和body指定邮件的标题和内容，cc是抄送地址，bcc是密件抄送地址多个抄送人使用;分割。

alt是图像的描述性本文，在图片不可见或者下载不成功的时候可看到该属性的指定文本。
