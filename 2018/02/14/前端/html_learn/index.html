HTML的基础标签认识

按照字母顺序排列(from w3school)

  • 注释
  • 定义文档类型
  • a 定义锚，也就是超链接
  • abbr 标记缩写标签
  • address 联系信息
  • area
    标签定义图像映射中的区域

必须属性:

可选属性

  • article
    规定独立的自包含内容

  • aside
    定义其所处内容之外的内容。

  • audio
    定义声音，比如音乐或其他音频流。

属性：

  • b
    标签规定粗体文本。
  • base
    指定为页面上的所有链接规定默认地址或默认目标。即当对应的标签有href和target这两个属性的时候（比如<a>，<img>），base标签可以给他们默认指定，base标签只能存在head标签之中。
  • bdo
    可覆盖默认的文本方向。比如之前的文本是从左到右顺序变为从右到左,r-t-l 或者是l-t-r

  • big
    呈现大号字体效果。

  • blockquote
    该标签定义块引用

    与
    之间的所有文本都会从常规文本中分离出来，经常会在左、右两边进行缩进（增加外边距），而且有时会使用斜体。也就是说，块引用拥有它们自己的空间。

  • body
    一个简单的 HTML 文档，带有最基本的必需的元素：
  • br
    换行
  • button
    定义一个按钮。始终需要设置一个type属性。因为不同的浏览器type不同。

属性：

  • canvas
    定义图形，比如图表和其他图像。他只是图像容器，必须使用脚本绘制图形。包含height和width。
  • caption
    定义表格标题，必须紧跟table标签之后，一个表格只能定义一个标题，通常这个标题会被居中表格之上。
  • cite
    通常表示它所包含的文本对某个参考文献的引用，比如书籍或者杂志的标题。默认以斜体显示
  • code
    用于表示计算机源代码或者其他机器可以阅读的文本内容。
  • col
    为表格中一个或多个列定义属性值，按照顺序。只能在table和colgroup元素使用col标签。

属性

  • colgroup
    用于对表格中的列进行组合，以便对其进行格式化，只能在 table 元素中使用。
  • datalist
    定义选项列表。与input元素配合使用，来定义input可能的值。
  • dd
    在定义列表中定义条目的定义部分。
  • del
    删除线
  • dfn
    可标记那些对特殊术语或短语的定义。现在流行的浏览器通常用斜体来显示 中的文本。将来， 还可能有助于创建文档的索引或术语表。
  • div
    可定义文档中的分区或节，
    标签可以把文档分割为独立的、不同的部分。它可以用作严格的组织工具，并且不使用任何格式与其关联。如果用 id 或 class 来标记
    ，那么该标签的作用会变得更加有效。
  • dl
    定义了定义列表,用于结合
    （定义列表中的项目）和
    （描述列表中的项目）。
  • dt
    定义了定义列表中的项目（即术语部分）
  • em
    斜体
  • embed
    定义嵌入的内容，比如插件。

属性

  • fieldset
    可将表单内的相关元素分组。

属性

  • figcaption
    定义figure元素的标题，位于figure的第一个或者是最后一个子元素的位置。
  • figure
    规定独立的流内容（图像、图表、照片、代码等等）
  • font
    规定文本的字体、字体尺寸、字体颜色。但是一般使用css来实现。
  • footer
    文档中的页脚部分
  • frame
    frameset 中的一个特定的窗口（框架）。

属性：

  • frameset
    可定义一个框架集。它被用来组织多个窗口（框架）。每个框架存有独立的文档。在其最简单的应用中，frameset 元素仅仅会规定在框架集中存在多少列或多少行。您必须使用 cols 或 rows 属性。
  • h1...h6
    标题
  • head
    一个简单的 HTML 文档，带有最基本的必需的元素：
  • header
    定义文档的页眉（介绍信息）
  • hr
    水平线
  • html
    此元素可告知浏览器其自身是一个 HTML 文档。
  • i
    显示斜体文本效果
  • iframe
    会创建包含另外一个文档的内联框架（即行内框架)，可以把需要的文本放置在 <iframe> 和 </iframe> 之间，这样就可以应对无法理解 iframe 的浏览器。
    属性
  • img
    向网页中嵌入一幅图像

必需的属性是src和alt，alt是加载失败的时候显示的text
属性

  • ins
    定义已经被插入文档中的文本。是一个下划线
  • kbd
    定义键盘文本
  • keygen
    规定用于表单的密钥对生成器字段,当提交表单时，私钥存储在本地，公钥发送到服务器。
  • legend
    为 fieldset 元素定义标题
  • li
    标签定义列表项目
  • link
    定义文档与外部资源的关系，最常见的用途是链接样式表。

属性

  • main
    规定文档的主要内容。在一个文档中，不能出现一个以上的 <main> 元素。<main> 元素不能是以下元素的后代：<article>、<aside>、<footer>、<header> 或 <nav>。
  • map
    定义一个客户端图像映射。图像映射（image-map）指带有可点击区域的一幅图像。

他的id是必须的，与image的usemap对应，属于客户端的图像映射。

  • mark
    定义带有记号的文本。
  • meta
    可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词，位于文档的头部，不包含任何内容。 标签的属性定义了与文档相关联的名称/值对。
  • meter
    定义已知范围或分数值内的标量测量。也被称为 gauge（尺度）

属性

  • nav
    定义导航链接的部分
  • noframes
    可为那些不支持框架的浏览器显示文本。noframes 元素位于 frameset 元素内部。
  • noscript
    定义在脚本未被执行时的替代内容（文本）
  • object
    定义一个嵌入的对象。请使用此元素向您的 XHTML 页面添加多媒体。此元素允许您规定插入 HTML 文档中的对象的数据和参数，以及可用来显示和操作数据的代码。
    <object> 标签用于包含对象，比如图像、音频、视频、Java applets、ActiveX、PDF 以及 Flash。
    object 的初衷是取代 img 和 applet 元素。不过由于漏洞以及缺乏浏览器支持，这一点并未实现。
    浏览器的对象支持有赖于对象类型。不幸的是，主流浏览器都使用不同的代码来加载相同的对象类型。
    而幸运的是，object 对象提供了解决方案。如果未显示 object 元素，就会执行位于 <object>和 </object> 之间的代码。通过这种方式，我们能够嵌套多个 object 元素（每个对应一个浏览器）。<param> 标签定义用于对象的 run-time 设置。不要对图像使用 <object> 标签，请使用 <img> 标签代替。

属性

  • ol
    有序列表
  • optgroup
    标签定义选项组
  • option
    选择列表item
  • output
    定义不同类型的输出，比如脚本的输出

属性：

  • p
    段落
  • param
    允许您为插入 XHTML 文档的对象规定 run-time 设置，也就是说，此标签可为包含它的 <object> 或者<applet>标签提供参数。

属性

  • pre
    可定义预格式化的文本，被包围在 pre 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。
  • progress
    标示任务的进度（进程）
  • q
    短引用
  • s
    定义加删除线文本定义
  • samp
    表示一段用户应该对其没有什么其他解释的文本字符。要从正常的上下文抽取这些字符时，通常要用到这个标签。

  • script
    插入一段js

  • section
    定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。

  • select
    创建单选或多选菜单，<option> 标签用于定义列表中的可用选项。

属性

  • small
    呈现小号字体效果
  • source
    为媒介元素（比如
  • span
    组合文档中的行内元素
  • strike
    可定义加删除线文本定义，不过建议使用<del>替换
    *strong 强调语气，加粗字体
  • style
    用于为 HTML 文档定义样式信息。
  • 定义下标文本
  • sup
    定义上标文本
  • table
    定义 HTML 表格，简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。更复杂的 HTML 表格也可能包括 caption、col、colgroup、thead、tfoot 以及 tbody 元素

属性

  • tbody
    表示表格主体（正文）。该标签用于组合 HTML 表格的主体内容。tbody 元素应该与 thead 和 tfoot 元素结合起来使用。thead 元素用于对 HTML 表格中的表头内容进行分组，而 tfoot 元素用于对 HTML 表格中的表注（页脚）内容进行分组。

属性

  • td
    tbody 元素中内容的垂直对齐方式
  • textarea
    多行文本输入控件。

属性

  • tfood
    与tbody类似，定义表格的页脚（脚注或表注）
  • th
    表头单元格
  • thead
    表格的表头
  • title
    title
  • tr
    表格中的行
  • u
    添加下划线
  • ul
    无序列表
  • var
    定义变量。您可以将此标签与 <pre> 及 <code> 标签配合使用。
  • video

定义视频，比如电影片段或其他视频流。
