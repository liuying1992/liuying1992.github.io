css的布局模型

布局模型是建立在盒子模型的基础之上的，有三种

  1. 流动性(flow)
  2. 浮动性(float)
  3. 层模型(Layer)
Flow布局模型

他是网页默认状态下的布局模型，他的特征是：块状元素都会在所处的包含元素内自上而下顺序垂直延伸分布，因为默认状态下块状元素宽度都是100%。而内联元素则是会在所处的包含元素才从左到右水平分布显示。

Float浮动模型

他是可以设置标签浮动显示，即有点像相当于(relative)的感觉。他可以设置两个块状元素在同一行而不是失去块状元素的其他特点。

当当前窗口足够大的时候，他会顺序在左边排列。div2在div1右边是因为div1已经占据了left的一部分位置。

Layer 层模型

有点像FrameLayout。他有三种形式

  1. 绝对定位(position；absolute)
  2. 相对订单(position:relative)
  3. 固定定位(position:fixed)

  • absolute
    他会将元素从文档流中拖出来，然后使用left，right，top，bottom属性相对于其最接近的一个具有定位属性的父包含块进行绝对定位。如果不存在，则以body元素为父包含。

  • relative
    他通过left，right，top，bottom属性确定元素在正常文档流中的偏移位置。首先是按照static(float)方式生成一个元素，然后相对于旧位置开始移动。

  • fixed
    他跟absolute相似，但是他的相对位移是网页窗口，他是不会碎浏览器窗口滚动而变化的。(常见的网页广告)

  • relative 和absolute的一起使用
    他们二者结合一起，可以做到某一个元素相对于另外一个元素进行定位。但是有两个要求：

  1. 参照定位的元素必须是相对定位元素的父元素
  2. 参照定位元素必须加入position:relative;
  3. 定位元素必须加入position:absolute;
