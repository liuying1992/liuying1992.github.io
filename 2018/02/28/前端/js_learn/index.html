写入JavaScript

  • 插入js

a. 在html代码文件中，可以添加script标签，可以在head也可以在body中添加。

b. 可以引入独立的js文件，也是通过script标签

  • js的解析顺序
    js的代码可以放在html中的任意位置，但是 由于html是按照先后顺序，从上而下执行的，所以前面的script会被先执行，后面的就后执行。所以，一些初始化的js需要放在head标签当中，非初始化的js可以放在body当中。

  • js分行
    js的换行，即一条语句结束的时候，可以不写分号，建议还是写上。

  • js注释
    有两种，一种是//单行注释，另外一种是/**/多行注释

  • js变量
    变量一般要先声明再赋值，同时可以重复赋值。区分大小写，注意html是不区分大小写的。命名必须是字母下划线美元$符号开头。不能使用js关键字。

  • 函数

JavaScript当中，所有的事物都是对象，比如字符串，数值，函数等，每个对象都具有属性和方法。

浏览器对象

  • window对象是指当前的浏览器窗口

  • setInterval(code,交互时间)
    他的计时单位是毫秒，具有一个返回值，使用改返回值可以用在clearInterval()中停止重复执行。

  • setTimeout(code,delay)
    延迟一段时间之后去执行代码

  • History对象
    他记录了用户曾经浏览过的页面，并且可以实现浏览器的前进和后退功能，通过history.[property|method]进行访问。

  • Location对象
    用户获取窗体的URL，通过location.[property|method]访问

  • Navigator对象
    它包含浏览器的信息，通常用来检测浏览器与操作系统版本

  • screen对象
    用户获取屏幕信息
