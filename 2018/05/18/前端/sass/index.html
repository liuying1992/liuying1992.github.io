Sass

发表于 2018-05-18   |   分类于 前端

SCSS

Sass是一门高于CSS的元语法，他有着比CSS更加强大的功能，能够提供更简洁，更优雅的语法，同时提供多种功能来创建可维护和管理的样式表。
目前有的css预处理器有：

  1. Sass(SCSS)
  2. Less
  3. Stylus
  4. Turbine
  5. Swithch CSS
  6. CSS Cacheer
  7. DT CSS

主流是Sass和Less以及Stylus

SCSS的基本概念与安装

SCSS其实是跟Sass是同一个东西，都称之为Sass，他们有两点不同

  1. 文件扩展名不同。Sass以.sass结尾，SCSS以.scss结尾
  2. 语法书写方式不同，Sass是以严格缩进式的语法规则，不带大括号。而SCSS的语法书写跟我们平常的CSS很相似。

Sass的安装
对于Mac：sudo gem install sass ，对于Windows，需要先安装ruby，然后在安装sass：gem install sass

Sass的语法

我们安装了Sass之后，可以编写Sass文件了，编写的时候，可以编写.sass或者.scss。其中.sass是严格缩进的，.scss是跟CSS类似的。所以我们一般使用.scss的就好。

Sass继承

  • Sass可以通过@extends实现样式继承。

  • 占位符
    占位符%，他只有在被@extend引用的时候才会起效，否则不会被编译到css文件中。而且，当被引用的时候，他会产生代码，同时将相同的代码合并一起。

对于混合宏，继承，占位符的对比：

  1. 混合宏不会合并参数，但是能传值。当需要传值的时候使用。
  2. 继承，会合并，不能传值，当该值已经定义的时候使用这种。
  3. 占位符，会合并，当代码没有基类存在，可以使用

Sass数据类型

数字，字符串，颜色，布尔值，空值，值列表(1 em 50% 之类的)。SassScript也支持其他CSS属性，但是他不会特殊对待，只是一律视为无引用号字符串。

  2. 无引用号 bold

Sass的运算

  • 加法
    在变量和属性中都可以做加法运算，但是对于不同类型的单位一起时，在Sass中会报错。

  • 减法
    与加法类类似

  • 乘法
    他只能其中一个值由数值单位，不能两个同时有，而且需要类型相同

  • 除法运算
    他与乘法类似，但是他需要在两边的除数和被除数中添加一个括号。

  • 颜色运算
    对于颜色计算，也支持上面几种，但是注意颜色他是按照argb分段计算的，也就是一段一段结果计算完成再去拼接。

    color:#010203 * 2;//#020406;
    }
