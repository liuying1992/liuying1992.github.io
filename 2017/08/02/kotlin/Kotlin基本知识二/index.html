发表于 2017-08-02   |   分类于 kotlin

面向对象

类构造器和继承：

类定义也是使用class关键字，构造器使用constructor关键字，里面定义属性。继承类使用:，需要写出父类的属性和自己构造器的属性。父类的属性使用它的构造器的参数名既可，类需要被继承需要加上open关键字，否则是final的类不能被继承。

普通属性和组合属性

普通属性就是一般的构造器中的属性，组合属性就是通过计算得到值的属性。

get和set是关键字，通过get为该组合属性复制，通过set改变其他属性。当没有使用set的时候，组合属性类型为val。

子类中的属性和属性覆盖

子类可以定义自己特有的属性
当父类定义了一个属性，子类也是定了了同名的属性的时候，父类属性需要加入open关键字，子类属性需要加上override关键字。
