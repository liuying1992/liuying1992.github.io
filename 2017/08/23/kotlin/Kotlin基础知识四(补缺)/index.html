关于get/set：

  1. get()是设置对象获取该属性时候的返回
  2. set是通过该属性的值可以对其他属性或者方法进行设置或者调用。他被调用的时间是该对象对该属性赋值的时候
  3. 对于val属性他是只有get没有set的
  4. 在get方法中不可以再次调用本身属性，否则会栈溢出。

  1. 只能对应在类外部或者object(伴生对象或者对象表达式)内部
  2. 只能使用String或者是原生的数据类型(Int，Double等)
  3. 不能自定义getter(因为调用的时候直接调用或者内部为我们生成了，所以不需要)

在Kotlin使用注解的时候，注解参数中的属性，只能是编译期常量(其他形式的属性都不能使用在注解参数中)
