TypeAdapter

使用TypeAdapter的时候，他的优先级是最高的，这时候可以不用任何的注解，任何的注解都会失效。定义adapter：

他有两个方法，一个是write(JsonWriter out, User value)，用于格式化成json的时候，另一个是 read(JsonReader in) ，用于解析Json的时候。如何格式化和如何解析都是我们自己定义的。

关于注册的继承：

registerTypeAdapter必须使用包装类型，所以int.class,long.class,float.class和double.class是行不通的。同时不能使用父类来替上面的子类型，这也是为什么要分别注册而不直接使用Number.class的原因。换成registerTypeHierarchyAdapter 就可以使用Number.class而不用一个一个的当独注册啦！
